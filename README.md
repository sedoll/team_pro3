# 📂 프로젝트 소개 
​
안녕하세요! <br/>
**Tspoon**은 학부모들을 위한 온라인 커뮤니티 플랫폼으로,<br/>
양육과 교육에 관한 정보를 공유하고 상호 지원하는 공간을 제공합니다.  
​<br/><br/>
학부모들이 자녀의 성장과 교육 과정을 더욱 효과적으로 관리하고,<br/> 
다른 학부모와 경험을 나누며, <br/>유익한 자원을 활용하여
 **함께 성장할 수 있는 플랫폼을 제공**하는 것이   
 저희 팀의 기획의도 입니다.  
 <br/> <br/>
